# React Projects

> A collection of React applications and experiments built to learn and practice React.js concepts.

## 🚀 About
This repository contains multiple React projects I’ve have put in a single website I’ve built to strengthen my frontend development skills. Each project focuses on practical UI features, state management, API usage, component structure, and modern React best practices.

## 📦 Projects
Each folder contains a separate level of project:
- **Easy Projects**

- **Project 1 Accordian** – Description (It handle accordians. By default  single accordian will open and on click of button you can open       multi. accordian at same time).  
- **Project 2 Color Generator** – . Description (It creates a random color on click of a button and you can change it from Hex to Rgb color type). 
- **Project 3 Rating** – Description (It allows you rate. on hover you can select range and on click it will store the value and rate it a ccordingly).  
- **Project 4 Theme Changer** – Description (It allows you to change theme from dark to light and it also stores the value).  
- **Project 5 Tabs** – Description (it allows you to change tabs in oder to see the content of it). 

- **Medium Projects**

- **Project 6 Scrool indicator** – Description (It basicaly stores and allow you to see how much you have scrolled the data or page).  
- **Project 7 Modal Popup** – Description (It allows you to open model in order to see its content).  
- **Project 8 Search** – Description (It is a search functionality which allows you to search from api data).  
- **Project 9 Tic Tac Toe** – Description (It is a game with all functionality of winning , lossing and draw. You can also reset or start new game).  
- **Project 10 Scroll & Size** – Description (In it data is show from api and screen size (height,Width) is shown and two buttons which allows you to scroll to top and bottom of the page).  
- **Project 11 outside Click Close** – Description (It allows to show or open model on click of button and in oder to close the modal you have to click outside of the modal).

- **Hard Projects**

- **Project 12 Food Recipe App** – Description (It's a website where you can see a navbar with search functionality and data from api you can view there detailed recipe and can move to favorites).  
- **Project 13 React Redux** – Description (it a shopping website where data is called from api and shown in cards form and you can add items to cart with the help of react redux toolkit).   


## 🛠️ Tech Stack
This repo uses:
- **React.js** (for UI)  
- **JavaScript / JSX**  
- **CSS / Tailwind / DaisyUI** (depending on project)  
- **React Router** (for routing in some apps)  
- **React Icons** (for icons)  
- **React Redux** (to cart functionality)  
- **External APIs** (where applicable)  


## ⚙️ Installation & Setup
To run any project locally:

1. Clone the repository  
   ```bash
   git clone https://github.com/AnujHub03/react-projects.git

2. Navigate into the specific project folder
   cd react-projects/project-name

3. Install dependencies
   npm install

4. Start the development server
   npm start

5. The app should open at http://localhost:3000
